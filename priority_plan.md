# 📋 Ежедневный план: B1 → B2 (формат приоритетов)

## 🎯 ОБЯЗАТЕЛЬНЫЙ МИНИМУМ (делай каждый день)

### ✅ Приоритет 1: Anki повторение (первым делом!)
**Что:** Повторяешь карточки, которые Anki показывает на сегодня  
**Сколько:** Обычно 20-50 карточек (10-15 минут)  
**Важно:** Делай ЭТО ПЕРВЫМ, даже если больше ничего не успеешь

---

### ✅ Приоритет 2: Murphy — 1 юнит грамматики
**Что:** Изучаешь 1 юнит из English Grammar in Use  
**Как:**
1. Читаешь правило на левой странице (5 мин)
2. Делаешь ВСЕ упражнения на правой странице ПИСЬМЕННО (10-15 мин)
3. Проверяешь по ключам, исправляешь ошибки (5 мин)

**Результат:** 1 юнит в день = за 5 месяцев пройдёшь весь Murphy (145 юнитов)

**Важно:** Если юнит слишком лёгкий (всё знаешь) — можешь пропустить и сразу перейти к следующему.

---

## 📚 ЖЕЛАТЕЛЬНО (если есть время и силы)

### ✅ Приоритет 3: English File — 1 урок (File)
**Что:** Проходишь 1 File из English File Intermediate  
**Структура урока:**
1. **Vocabulary** — изучаешь новые слова, записываешь незнакомые (5-7 мин)
2. **Reading или Listening** — читаешь текст или слушаешь аудио (10 мин)
3. **Grammar** — изучаешь грамматику, делаешь упражнения (10 мин)

**Темп:** 1 File за 1-2 дня. В книге ~10 Files = закончишь учебник за 10-20 дней.

**Важно:** НЕ пропускай аудио — это критически важно для listening.

---

### ✅ Приоритет 4: Workbook — упражнения
**Что:** Делаешь упражнения из Workbook к пройденному File  
**Сколько:** Все упражнения к одному уроку (15-20 мин)  
**Важно:** Проверяй ответы по ключам. Если ошибки — переделывай.

**Зачем:** Без Workbook грамматика и слова не закрепятся.

---

## 🌟 ДОПОЛНИТЕЛЬНО (если остались силы)

### ✅ Приоритет 5: Добавление новых слов в Anki
**Что:** Добавляешь слова, которые выписал за день  
**Лимит:** Максимум 10-15 новых слов  
**Как:**
1. Собираешь слова за день в файл
2. Отправляешь мне список или скриншот
3. Я создаю файл для Anki
4. Ты импортируешь (File → Import)

**Важно:** НЕ добавляй сразу 50-100 слов — это перегруз. Anki покажет новые слова завтра для изучения.

---

### ✅ Приоритет 6: Input (YouTube/подкаст/чтение)
**Что:** Смотришь/слушаешь что-то короткое на английском  
**Варианты:**
- **YouTube:** BBC 6 Minute English (6 мин), English with Lucy (10-15 мин)
- **Подкасты:** Luke's English Podcast (слушай в дороге)
- **Чтение:** News in Levels (newsinlevels.com) — короткая статья (5-7 мин)

**Цель:** Привыкание к звучанию языка, расширение лексики в контексте.

---

## 🎬 ВЫХОДНЫЕ (субботы-воскресенья)

### Суббота: Фильм/сериал на inoriginal.ru
**Что:** Смотришь 1 эпизод сериала с двойными субтитрами  
**Как:**
1. Смотришь с англ + рус субтитрами
2. Кликаешь на незнакомые слова, добавляешь в словарь (максимум 15-20 слов!)
3. После просмотра — скриншот словаря → отправляешь мне
4. Я создаю файл для Anki → ты импортируешь

**Важно:** НЕ пытайся выписать все слова — только самые важные!

---

### Воскресенье: Повторение и Speaking
**Что делаешь:**
1. **Anki** — прогоняешь все карточки за неделю (15 мин)
2. **Murphy Review** — просматриваешь юниты за неделю, переделываешь ошибочные упражнения (15 мин)
3. **Speaking Practice** — записываешь себя на диктофон (15 мин):
   - Пересказываешь текст из English File своими словами
   - Или описываешь свой день на английском
   - Слушаешь запись, ищешь ошибки

**Цель:** Закрепление материала за неделю. Без повторения всё забудется.

---

## ✅ Что делать СЕЙЧАС

1. **Установи Anki:** ankiweb.net (бесплатно для Windows/Mac/Android)
2. **Скачай готовую колоду Murphy:**
   - Открой: https://ankiweb.net/shared/info/1337411742
   - Нажми "Download"
   - Открой Anki → File → Import → выбери скачанный файл
3. **Начни с минимума:**
   - Завтра утром: Anki 10 мин + Murphy Unit 1 (20 мин)
   - Послезавтра: то же самое
   - Через неделю добавь English File

**НЕ пытайся сразу делать всё** — начни с Приоритетов 1 и 2, остальное добавишь, когда войдёшь в ритм.

## 📊 Отслеживание прогресса

### Каждую неделю проверяй:
- **Anki:** Сколько слов выучено (статистика в программе)
- **Murphy:** Сколько юнитов пройдено (цель: 5-7 юнитов/неделю)
- **English File:** Сколько Files пройдено (цель: 3-5 Files/неделю)

### Каждый месяц:
- Проходи тест на уровень: **efset.org**
- Цель: За 3 месяца B1 → B1+, за 6 месяцев → B2

## 💬 Как работать со мной

**Когда отправлять слова для Anki:**
- После каждого File из English File (10-15 слов)
- После просмотра эпизода на inoriginal (10-15 слов)
- 1-2 раза в неделю, когда накопится 20-30 слов

**Что отправлять:**
- Просто список слов (один под другим)

**Что получишь:**
- Файл `words.txt` с переводами, примерами, транскрипциями
- Импортируешь в Anki за 30 секунд
